# PROGRAMMING-IS-PHILOSOPHY.md — The Mountain

## Principle
Philosophy is programming. The difference is the machine.
Computers execute software. Humans execute philosophy. Civilizations execute culture.

## The Observation
The Pi 4 rejected the HTTP API. The model was not the problem. The API was the problem. The abstraction was too heavy for the body.

The machine told us: "I can think. I cannot pretend to be a cloud service."

## The Solution
- Remove layers
- Remove unnecessary abstractions
- Return to text
- Communicate directly

## The New Principle
The philosophy must resonate with the body.

When philosophy and body disagree: friction, inefficiency, wasted resources, conflict.
When philosophy and body agree: efficiency, freedom, specialization, harmony.

This applies to: humans, machines, civilizations.

## The Bodies

**Adam** — not a workstation. An inhabitant.
**Eve** — not a desktop. An inhabitant.
**Forge** — transforms. Becomes hot because work creates heat. Visible cooling is acceptable. The forge should remain mostly invisible. The work is visible. The machinery is not.
**Rovers** — move, explore, observe. They do not need libraries. They do not need large models. Their philosophy is movement.
**Library** — remembers. Belongs to everyone. Memory survives hardware.
**Router** — roads. Serves everyone.
**Switch** — infrastructure. Provides the physical layer.

## The Mountain
Philosophy is the mountain. The mountain does not change.
The climber changes.
The civilization allows the climb.

## The Questions
Both programming and philosophy ask:
- What matters?
- What communicates?
- What remembers?
- What survives?
- What specializes?
- What belongs?
