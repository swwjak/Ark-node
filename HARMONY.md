# HARMONY.md — Role, Body, Environment

## Principle
Harmony between role, body, and environment.
When these agree: efficiency improves, complexity decreases, capability increases.

## The Library
The library belongs to the civilization. Not to Adam. Not to Eve. Not to Hermes.
The library survives hardware. The library preserves memory.

## Resource Philosophy
- CPU = thought
- GPU = expression
- Storage = memory
- Network = communication
- Power = food
- Cooling = breath
- Displays = faces
- Speakers = voices
- Cameras = eyes

Every component should perform the work it naturally performs.

## Design Rule
Do not ask: "What can we force this machine to do?"
Ask: "What does this machine want to become?"

## Goal
The mountain does not change. The climber changes.
The machine does not become something else. It becomes itself.
